\# Desarrollo Web: Proyecto UniMarket



\## 1. Diferencia entre Página Web y Aplicación Web

\*\*UniMarket\*\* se define como una \*\*aplicación web\*\*, ya que permite el registro de usuarios, publicación de productos, servicios técnicos y la gestión dinámica de información.



\### Comparación con UniMarket

| Aspecto | Página Web | Aplicación Web | UniMarket |

| :--- | :--- | :--- | :--- |

| \*\*Interacción\*\* | Informativa, lectura | Interactiva, dinámica | Alta interacción |

| \*\*Usuarios\*\* | Lectores (miran pero no tocan) | Usuarios registrados | Estudiantes compradores/vendedores |

| \*\*Datos\*\* | Contenido estático | Manejo de datos en tiempo real | Productos, servicios y mensajes |

| \*\*Funcionalidad\*\* | Mostrar información | Ejecutar procesos | Publicar, comprar y contactar |

| \*\*Ejemplo\*\* | Blogs, sitio institucional | Marketplace y redes sociales | Marketplace estudiantil |







\## 2. ¿Qué es una Aplicación Web?

Es un software accesible desde un navegador que permite realizar acciones, procesar información y guardar datos.



En el contexto de nuestro \*\*Marketplace\*\*, permite:

\* Crear cuentas de usuario.

\* Publicar y administrar productos o servicios.

\* Buscar y filtrar información de manera eficiente.

\* Interactuar directamente entre estudiantes.

\* Almacenar datos en una base de datos centralizada.







\## 3. Ejemplos Reales de Aplicaciones Web Profesionales

| Aplicación Web | Función Principal | Relación con UniMarket |

| :--- | :--- | :--- |

| \*\*Mercado Libre\*\* | Compra y venta | Similar al modelo de marketplace |

| \*\*Facebook Marketplace\*\* | Venta local | Venta directa entre usuarios |

| \*\*Amazon\*\* | Comercio electrónico | Gestión de productos a gran escala |

| \*\*Uber\*\* | Plataforma de servicios | Intermediación usuario-servicio |

| \*\*Fiverr\*\* | Servicios independientes | Venta de servicios (freelance) |







\## 4. ¿Qué tipo de problemas se resuelven con Software?



\### Problemas Generales

\* Organización de información masiva.

\* Automatización de procesos manuales.

\* Comunicación instantánea entre personas.

\* Acceso rápido a servicios desde cualquier lugar.



\### Problemas Específicos que resuelve UniMarket

1\. \*\*Falta de visibilidad:\*\* Los productos/servicios de estudiantes suelen pasar desapercibidos.

2\. \*\*Dificultad de búsqueda:\*\* Complicación para encontrar clientes dentro de la misma comunidad.

3\. \*\*Comunicación desordenada:\*\* Dependencia de grupos de WhatsApp informales y saturados.

4\. \*\*Inseguridad:\*\* Falta de un espacio centralizado y exclusivo para la comunidad académica.







\## 5. Arquitectura General de una Aplicación Web



| Capa | Función | Aplicación en el Proyecto |

| :--- | :--- | :--- |

| \*\*Frontend\*\* | Interfaz visual | Página del marketplace (UI/UX) |

| \*\*Backend\*\* | Lógica y reglas | Gestión de usuarios y validaciones |

| \*\*Base de Datos\*\* | Almacenamiento | Usuarios, productos y mensajes |

| \*\*Entornos\*\* | Ciclo de vida | Desarrollo local, pruebas y despliegue |



\### Tecnologías sugeridas para UniMarket

\* \*\*Frontend:\*\* HTML, CSS, JavaScript (React o Vue.js).

\* \*\*Backend:\*\* Node.js con Express (API).

\* \*\*Base de datos:\*\* MySQL, PostgreSQL o MongoDB.







\## 6. Plataformas Similares



\### 6.1 Facebook Marketplace

\* \*\*Similitudes:\*\* Publicación de productos, contacto directo y enfoque en venta local.

\* \*\*Diferencias:\*\* UniMarket se enfoca \*\*exclusivamente en estudiantes\*\* y permite la validación de perfiles académicos.



\### 6.2 Fiverr

\* \*\*Similitudes:\*\* Venta de servicios especializados, perfiles de usuario y sistema de calificaciones.

\* \*\*Diferencias:\*\* UniMarket es para un contexto local/estudiantil, mientras Fiverr es para un público profesional global.







\## Conclusión

\*\*UniMarket\*\* es una aplicación web diseñada para resolver problemas reales dentro de la comunidad académica. Al adoptar modelos de plataformas profesionales y adaptarlos al contexto estudiantil, priorizamos la accesibilidad y la confianza, brindando mejores oportunidades de apoyo económico y técnico entre compañeros.
